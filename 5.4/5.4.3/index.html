### 5.4.3 1) Сделать обьект, у которого будет свойство twoArray и метод create2Array. У метода будет два параметра: количество айтемов в массиве (внешний и внутренний) и значение айтемов. Метод создает двумерный массив и присваивает его свойству twoArray текущего обьекта. Сделать без использования метода push. create2Array(3, 'hi') создаст: [ ['hi', 'hi', 'hi'], ['hi', 'hi', 'hi'], ['hi', 'hi', 'hi'] ] 2) К обьекту добавить свойство threeArray и метод create3Array. У метода будет два параметра: количество айтемов в массиве и значение айтемов. Метод создает трехмерный массив и присваивает его свойству threeArray текущего обьекта. Сделать с использованием метода push. create3Array(3, 'hi') создаст: [ [ ['hi', 'hi', 'hi'], ['hi', 'hi', 'hi'], ['hi', 'hi', 'hi'] ], [ ['hi', 'hi', 'hi'], ['hi', 'hi', 'hi'], ['hi', 'hi', 'hi'] ], [ ['hi', 'hi', 'hi'], ['hi', 'hi', 'hi'], ['hi', 'hi', 'hi'] ] ]
