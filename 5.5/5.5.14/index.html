### 5.5.14 Сделать объект с необходимыми методами. Главному методу будет передаваться массив с объектами, а он будет возвращать новый массив с отсортированными по убыванию объектами. Сортировать по свойству year. Добавь здесь еще один метод, который вернет строку имён разделенные запятыми, причем все латинские буквы должны быть заменены на русские, например "Tania" должна стать "Таниа" Массив с объектами: [ {name: "Tania", year: 1985}, {name: "Vlad", year: 1981}, {name: "Danik", year: 2013}, {name: "Artem", year: 2017} ] должен отсортироваться в такой вид: [ {name: "Artem", year: 2017}, {name: "Danik", year: 2013}, {name: "Tania", year: 1985}, {name: "Vlad", year: 1981} ]
