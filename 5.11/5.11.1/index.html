### 5.11.1 Напишите функцию formatDate(date), которая выводит дату date в формате дд.мм.гг: Например: const customDate = new Date(2014, 0, 30); // 30 января 2014 alert( formatDate(customDate) ); // '30.01.14' P.S. Обратите внимание, ведущие нули должны присутствовать, то есть 1 января 2001 должно быть 01.01.01, а не 1.1.1.
